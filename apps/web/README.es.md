# Verokodo Web

Idiomas: [English](./README.md) | [Español](./README.es.md)

Frontend en Astro para la experiencia inmersiva del oráculo de Verokodo.

## Comandos

- `bun run dev` - inicia el servidor de desarrollo de Astro
- `bun run build` - construye la salida estática
- `bun run preview` - previsualiza el sitio compilado

### Rutas

- `/` - entrada de nombre de usuario
- `/loading` - transición de carga y request al oráculo
- `/result` - profecía final y acciones para compartir

### Variable opcional

- `PUBLIC_API_BASE_URL` (por defecto: `http://localhost:3000`)
